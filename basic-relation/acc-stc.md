## STC

* English： Active Object Relation
* 中文： 受動者-動詞關係（vutta）
* 巴利语： vuttakamma - kiriyā

这里，宾语形式的表达某种空间或时间的词语被用来表达；任何一个因素（空间/时间）的持续性。
>例如；
>1. yojanaṃ dīgho = （一）由旬长
>- yojanaṃ gacchiti = 走（一）由旬
>- yojanaṃ maggo = (一)由旬（长）的路
