# 二次衍生-动词分词

## 构成方式

| | | | | | | | | | | |
| -- | -- | -- | -- | -- | -- | -- | -- | -- | -- |-- |
|  |   | | | | | | | 变格词尾 | | |
| | | + | anta<br>māna |正在 |  = | [现在分词]词干 | | 1. 主格<br> 2. 宾格 | | |
| | | + | ta<br>na |曾经，已经 |  = | 过去分词词干 | | 3. 属格<br> 4. 为格 | | |
| (接头词)| 字根 | | | | | | + | | => | 变格词 | 
| | | + | aniya<br>tabba<br>eyya | 应该，可被 | = | 未来分词词干 | | 5. 来源格<br> 6. 工具格 | | |
| | | + | ya | | = | 动名词词干 | | 7. 位置格<br> 8. 呼格 | | |

## 例子： 

**viharanto** 是
 viharanta（正居住） 的 阳性 单数 主格

| -- | --  |-- | -- | 
| vi | hara | anta | o |
|异，特别 | 运送 | 正在 | 阳性 单数 主格 |
